# Tradecoins
Fullstack project to learn react, typescript and rust.

It is an online crypto portfolio, where you can store info about your crypto assets.

### Tech stack:

#### server:
- rust
- actix-web
- sqlx
#### client:
- typescript
- react
#### database:
- postgres
- redis
